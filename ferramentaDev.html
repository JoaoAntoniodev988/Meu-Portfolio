Flex Box

O Flexbox (abreviação de Flexible Box Layout Module, ou Modelo de Caixa Flexível) é um modelo de layout unidimensional do CSS3.

A principal função dele é permitir que você distribua o espaço entre os elementos de uma página e alinhe-os de forma dinâmica, inteligente e, como o próprio nome diz, flexível — mesmo que o tamanho desses elementos seja desconhecido ou mude constantemente.

CSS Grid Layout

O CSS Grid Layout (ou simplesmente Grid) é o sistema de layout mais poderoso do CSS. Enquanto o Flexbox é unidimensional (focado em alinhar as coisas em uma única linha ou coluna), o Grid é bidimensional.Isso significa que ele permite controlar linhas e colunas ao mesmo tempo, dando total controle sobre a estrutura e o posicionamento dos elementos em uma página.
